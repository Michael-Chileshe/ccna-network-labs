# CCNA Network Projects
A portfolio of 15 hands-on networking labs built in Cisco Packet Tracer,
covering the full CCNA 200-301 exam curriculum.

## Lab Environment
| Component | Details |
|---|---|
| Simulator | Cisco Packet Tracer 8.x |
| Routers | Cisco 2911, 2901 |
| Switches | Cisco 2960, 3560 (L3) |
| Protocols | OSPF, STP/RSTP, LACP, 802.1Q, DHCPv4/v6, NAT/PAT |

## Skills Demonstrated
- Cisco IOS CLI configuration and troubleshooting
- VLAN segmentation and 802.1Q trunking
- OSPF dynamic routing (v2 and v3)
- Network address translation (NAT/PAT)
- Access control lists (standard, extended, named)
- Switch security (port security, DHCP snooping, DAI)
- IPv4 and IPv6 dual-stack configuration
- Wireless LAN configuration and security
- Network troubleshooting methodology
